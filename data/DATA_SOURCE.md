# Data Source

## Kaggle Medical Insurance Cost Dataset

This project uses the Kaggle Medical Insurance Cost Dataset.

### How to download

1. Visit Kaggle: https://www.kaggle.com/datasets/mirichoi0218/insurance
2. Accept the terms and download the dataset.
3. Unzip the archive.

### Expected columns

- age
- sex
- bmi
- children
- smoker
- region
- charges

### File location

Save the dataset as `data/insurance.csv`.
